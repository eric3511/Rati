# TypeScript 进阶指南

这个专栏面向已有 TS 基础的开发者，带你深入类型系统的底层原理和高级模式。

::tip{title="学习前提"}
本专栏假设你已经熟悉 TypeScript 基本类型、interface 和 type 的区别、以及泛型的基础用法。
::

## 适合人群

- 在项目中踩过类型体操坑的前端工程师
- 希望写出更安全、更灵活的类型工具库的开发者
- 准备阅读开源库 TS 类型源码的工程师

## 你将学到

1. **类型系统深入** — 结构化类型、协变逆变、条件分发
2. **泛型实战** — 约束、推断、工厂模式
3. **条件类型** — `infer` 关键字、递归类型
4. **设计模式** — 模板字面量类型、映射类型、Branded Type
