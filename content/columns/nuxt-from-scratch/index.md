# Nuxt 从零开始

这套专栏带你系统学习 Nuxt 4，涵盖从基础到进阶的全部内容。

::tip{type="info" title="关于专栏"}
本专栏采用循序渐进的方式，每章都配有完整的示例代码和实战项目。
::

## 适合人群

- 有 Vue 3 基础，想学习 Nuxt 全栈框架的开发者
- 希望理解 SSR / SSG / CSR 渲染模式的前端工程师

::card{title="学习前提" description="开始学习前需要掌握的基础知识" horizontal}
### Vue 3 基础
了解 Composition API、组件系统、响应式原理

### JavaScript/TypeScript
熟悉 ES6+ 语法，了解 TypeScript 基本类型
::

## 你将学到

1. 项目初始化与配置
2. 基于文件系统的路由
3. 数据获取与水合
4. 中间件与插件
5. 部署与优化

::warning{title="环境要求"}
确保你的开发环境已安装 Node.js 18+ 和 pnpm/npm/yarn 等包管理器。
::

> 每章配有可运行的示例代码。
