# Nuxt 从零开始

这套专栏带你从零开始系统学习 Nuxt 4，从基础的脚手架搭建到生产环境部署，覆盖 Nuxt 全部核心概念。

::tip{title="关于专栏"}
本专栏采用循序渐进的方式，每章配有完整可运行的示例代码。所有代码基于 **Nuxt 4 + TypeScript + Tailwind CSS 4** 技术栈。
::

## 适合人群

- 有 Vue 3 基础，希望掌握 Nuxt 全栈框架的前端开发者
- 想深入理解 SSR / SSG / ISR 渲染模式的工程师
- 希望用 Nuxt 构建企业级应用的团队

## 你将学到

1. **项目初始化** — 理解 `nuxt.config.ts` 配置与模块系统
2. **文件路由** — 掌握 Nuxt 基于文件系统的动态路由
3. **数据获取** — 深入 `useFetch`、`useAsyncData` 与水合原理
4. **中间件与插件** — 编写可复用的中间件和插件
5. **渲染模式** — SSR、SSG、SWR 的选择与性能调优
6. **生产部署** — CI/CD、Docker、Vercel 多平台部署

::card{title="环境要求" description="开始学习前请确认以下环境就绪"}
### 必备环境
- Node.js 18+ 或 22+
- pnpm / npm / yarn（推荐 pnpm）
- VS Code + Volar 插件
::
